# Calculator — CodSoft Internship Task 3

This is a basic calculator I built as part of my Web Development Internship at CodSoft.

## What it does

It can do simple math — addition, subtraction, multiplication and division. There's also a clear button to reset everything and a backspace button if you press the wrong number.

## Files

- index.html — the structure
- style.css — the styling
- script.js — the logic

## How to run it

Just download the three files, keep them in the same folder and open index.html in your browser. No installs, no setup needed.

You can also use your keyboard — numbers and operators work directly, Enter for equals, Backspace to delete and Escape to clear.

## What I learned

I had never really used CSS Grid before this. Laying out the calculator buttons in a grid made a lot more sense than trying to do it with flexbox. The JavaScript part took me a bit to figure out, especially handling the backspace and making sure errors don't break the whole thing.
