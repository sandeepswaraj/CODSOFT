# Task 2 - Landing Page (Coffee Shop)

This is my second task for the CodSoft Web Development Internship (Level 1).

I made a landing page for a fictional coffee shop called **Brew & Co** using only HTML and CSS. No frameworks, no libraries — just plain code.

---

## What I Built

A simple one-page website for a coffee shop. It has a clean black and white design and works on both desktop and mobile.

### Sections included:

- **Navbar** — logo on the left, links on the right. Collapses to a hamburger menu on small screens.
- **Hero** — big heading, a short description, and a call-to-action button.
- **Features** — three cards explaining why someone would choose this coffee shop.
- **About** — a short paragraph about the shop's story.
- **Contact** — a simple form with name, email, and message fields.
- **Footer** — copyright text at the bottom.

---

## Files

```
Landing_Page/
├── index.html
└── style.css
```

- `index.html` — all the page structure and content
- `style.css` — all the styling, colors, layout, and responsive rules

---

## How to Run

1. Download or clone this folder
2. Open `index.html` in any browser
3. That's it — no setup needed

---

## What I Learned

- How to structure a multi-section landing page
- Using flexbox and CSS grid for layout
- Making a sticky navbar
- Writing media queries to make the page mobile-friendly
- Keeping a consistent color theme across the whole page

---

## Design Choices

I went with black and white because it looks clean and professional without needing too many colors. The font is Arial for body text and Georgia for headings to give it a slightly classic feel — like a real coffee shop menu would look.
